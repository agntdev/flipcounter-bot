# E1T2 — Implement /flip Command

**Weight:** 0.0000 (share of project budget)
**Reward:** 0 FLIP

Handle the /flip command to:
1. Identify the user by telegram_id (message.from.id)
2. Upsert user record (create or update, storing username if present)
3. Randomly select 'heads' or 'tails'
4. Insert a flip record in a transaction with:
   - flip record creation
   - flips_count increment
   - last_flip_at update
5. Reply with a message like '🎲 You flipped: Heads. Total flips: 7.'
6. Ensure responses are posted in the same chat where the command was issued

Write tests/specs/E1T2.json and append /flip to tests/commands.json
