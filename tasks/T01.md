# T01 — Scaffold the bot project

**Weight:** 0.0000 (share of project budget)
**Reward:** 0 FLIP

Scaffold the Telegram bot with grammY + the agnt bot toolkit (`createBot`): `package.json`, the TypeScript entrypoint, a `Dockerfile`, and a `.env.example` declaring `BOT_TOKEN`. The bot must boot and respond to an update.

HARD DELIVERABLE: create the `tests/specs/` directory (commit a `tests/specs/.gitkeep`) AND `tests/commands.json` (the command manifest). Every later feature task writes its OWN `tests/specs/<slug>.json` (a BotSpec dialog-test array) — NEVER a shared file — so concurrent feature PRs never conflict. The final go-live gate globs `tests/specs/*.json`, so this directory must exist from task one.
